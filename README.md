🎮 GameDev Module – GDG Game

This repository contains the Game Development (Unity) module for the GDG Game project. It includes all assets, scripts, and scenes required to build and run the game.

📌 Overview

The GameDev module focuses on:

Core gameplay mechanics
Scene design and level building
Player interaction systems
UI/UX for in-game experience
Integration with AI/dialogue systems (if applicable)

Built using Unity Engine, this module is responsible for rendering, physics, and real-time gameplay.

🛠️ Tech Stack
Engine: Unity
Language: C#
Version Control: Git + GitHub
Assets: Unity Asset System (Prefabs, Materials, Audio)
